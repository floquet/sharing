\subsection{\gdb}
\label{sec:gdb}
The application \refGdb \ is an open-source \href{https://www.onlinegdb.com/}{code debugger}. When codes are compiled with debug symbols (e.g. \texttt{gcc -d ...}) the debugger is a powerful.
