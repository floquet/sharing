\subsection{\env}
\label{sec:env}
A good starting point in debugging is to list environment variables with the command \refEnv \ to show where your application is looking for files and libraries. Most prominent are \href{https://en.wikipedia.org/wiki/Rpath}{\textt{RPATH}} and \href{https://unix.stackexchange.com/questions/46614/how-to-export-ld-library-path-to-all-users-and-system-services}{\textt{LD\_LIBRARY\_PATH}}. These variables can be manipulated using \href{https://linux.die.net/man/1/chrpath}{\textt{chrpath}} or \href{https://github.com/NixOS/patchelf}{\textt{patchelf}}.
